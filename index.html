    Terminal Based Maze Solver

    The "Terminal Based Maze Solver" project is a Python implementation featuring a maze generator creating random mazes with start (S) and end (E) points. It employs a depth-first search (DFS) algorithm to find a solvable path within the generated maze. The user interacts through a console interface, allowing options like printing the path or generating a new puzzle. The project is accessible via GitHub, has no external dependencies, and ensures a solvable path with a marked solution in the displayed maze.

    Tech stack : Python | Recursion | dsf-algorithm | Backtracking | 2d-arrays

    Typing Master

    "Typing Master" is an engaging Python-based typing test application designed for users of all skill levels. Featuring a user-friendly interface, diverse typing categories, and real-time performance tracking, it provides a fun and educational platform for improving typing skills. The leaderboard fosters competition, showcasing top performers, while immediate error feedback ensures accuracy. With easy installation and graceful exits, Typing Master offers a personalized and enjoyable typing practice experience.

    Tech stack : Python | Json | Randommodule | Time-module | List & Dictionaries

    Tkinter Calendar

    "Tkinter Calendar" provides a simple calendar application built using the Tkinter library in Python and the tkcalendar widget. It allows users to view and interact with a basic calendar, displaying the current month with the ability to navigate to previous and next months.
